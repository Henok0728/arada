# What is arada?
- arada is a web application that will help university students to learn new things by providing them with web simulation based projects of some of hardest courses in some degrees. It is a platform that will help students to master their skills and learn new things as the same time.

# How to use arada?
- You can access arada by visiting the website and logging in with your university credentials. Then you can select the course you want to learn and start the simulation. 

# Courses to Focus on 
- Electrical Engineering -> Electrical Machines and Signals and Systems
- Mechanical Engineering -> Thermodynamics and Fluid Mechanics
- Civil Engineering -> Structural Analysis
